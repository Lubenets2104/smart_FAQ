SmartTask User Manual

Руководство пользователя SmartTask

1. Начало работы
Зарегистрируйтесь на сайте smarttask.io и создайте первый проект.

2. Создание задачи
Нажмите "+ Задача" → введите название, дедлайн и назначьте исполнителя.

3. Управление проектом
Используйте доску Kanban для визуального контроля. Можно перетаскивать задачи между колонками.

4. Совместная работа
Комментируйте задачи, добавляйте файлы, упоминайте коллег через @.

5. Аналитика
Раздел "Отчёты" показывает выполненные задачи и загрузку команды.
