SmartTask Security & Privacy

Безопасность данных — приоритет №1 для SmartTask.

Хранение данных:
- Данные хранятся в дата-центрах AWS (Франкфурт, Германия)
- Резервное копирование каждые 6 часов
- Шифрование в состоянии покоя (AES-256)

Политика конфиденциальности:
- Мы не передаём данные третьим лицам без согласия клиента.
- Пользователь может удалить аккаунт и все данные по запросу.
- Cookie используются только для аутентификации.

Соответствие стандартам:
- GDPR
- ISO/IEC 27001
- SOC 2 Type II
